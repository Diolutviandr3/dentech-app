Tentang DENTECH

Prediksi Penyebaran DBD di DIY

DENTECH adalah aplikasi berbasis web yang dirancang untuk memprediksi penyebaran Demam Berdarah Dengue (DBD) di Provinsi Daerah Istimewa Yogyakarta. Aplikasi ini menggunakan data cuaca real-time (suhu, curah hujan, kelembapan, dan kecepatan angin) dari BMKG dan data historis dari BPBD untuk memberikan peringatan dini kepada pemerintah dan tenaga kesehatan.

Fitur Utama

  • Peta interaktif dengan warna risiko per kecamatan
  • Grafik distribusi risiko dan tren
  • Filter risiko untuk fokus pada wilayah tertentu
  • Notifikasi peringatan ketika risiko tinggi / sangat tinggi
  • Tabel data lengkap setiap kecamatan

Data Sumber

BMKG Suhu, Curah Hujan, Kelembapan, Kecepatan Angin (real-time simulasi)
BPBD Data historis penyebaran DBD di DIY
GeoJSON Batas administrasi kecamatan DIY dari sumber terbuka
